# Prompt pro Codex

Mám profilovou osobní stránku v Reactu + Tailwindu. Potřebuji ji dotáhnout do produkční podoby.

## Kontext
Jde o osobní/profesní web Martina Slívy. Web má sloužit jako profil pro seniorní AI leadership role — technické, produktové i konzultační. Má zdůraznit zkušenost napříč obory, silnou doménu v AI zdravotnictví, výuku, publikace a vybrané AI certifikace.

## Co chci udělat
1. Vyčisti sekci certifikací a ponech jen nejsilnější položky.
2. Ověř a oprav názvy kurzů podle přiložených certifikátů.
3. Doplň realistickou sekci prezentací s podporou PDF odkazů.
4. Zkontroluj mobilní layout a případné přetékání textu.
5. Navrhni drobný vizuální polish bez marketingového přestřelení.
6. Připrav to tak, aby komponent šel rovnou použít v moderním React/Next.js projektu s Tailwind CSS.

## Styl
- čeština,
- střízlivý a věrohodný tón,
- žádný hype,
- profil seniorního člověka, ne juniorské portfolio,
- méně self-promotion, více věcnost.
